# 💰 Expense Tracker System (DBMS Project)

A full-stack Expense Tracker application designed to help users manage and monitor their daily expenses efficiently. This project demonstrates database design, CRUD operations, and real-time expense tracking.

---

## 📌 Overview

The Expense Tracker allows users to:
- Add and categorize expenses  
- View expense history  
- Analyze spending patterns  
- Manage personal finances effectively  

---

## 🎯 Objectives

- Develop a system to record daily expenses  
- Store and manage data using a database  
- Perform CRUD operations (Create, Read, Update, Delete)  
- Provide insights into spending habits  

---

---

## 🗄️ Database Design

### Tables:
- **Users**
  - user_id (PK)
  - name
  - email

- **Expenses**
  - expense_id (PK)
  - user_id (FK)
  - amount
  - category
  - date
  - description
